<p align="center">

## 📖 Overview

**FoodShare** is a full-stack web application inspired by services like Uber Eats and Foodpanda, with a meaningful difference:  
It enables users to not only order and sell food, but also donate meals to people in need — promoting sustainability and community care.

## 🛠️ Main Technologies Used

- **React 19**
- **Vite** (Module bundler)
- **Tailwind CSS** & **DaisyUI**
- **Firebase** (Authentication)
- **React Query** (Data fetching)
- **Axios**
- **React Router**

---

## ✨ Key Features

- 🔐 Secure user authentication with Firebase
- 🍽️ Buy, sell, and donate food
- 📅 Track time and availability using `date-fns`
- ⚡ Real-time data loading with React Query
- 🎨 Fully responsive UI with TailwindCSS + DaisyUI
- 🎠 Image sliders and UI animations with React Slick
- 🧠 Smart alerts and popups with SweetAlert2

## 💻 Getting Started (Run Locally)

To run this project locally, follow these steps:

```bash
# Clone the repository
git clone https://github.com/Programming-Hero-Web-Course4/b11a11-client-side-rabiul3000.git

# Go to the project directory
cd b11a11-client-side-rabiul3000

# Install dependencies
npm install

# Start the development server
npm run dev
